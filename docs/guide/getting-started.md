# Getting Started

This guide will help you get started with using and contributing to KequTech Docs.

## Quick Start

1. Browse the documentation using the sidebar navigation
2. Use the search feature to find specific topics
3. Check out individual library guides for detailed information

## Contributing

Want to contribute to the documentation? Here's how:

1. Fork the repository
2. Create a new branch for your changes
3. Add or update documentation in the `docs` folder
4. Submit a pull request

## Local Development

To run the documentation locally:

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure
