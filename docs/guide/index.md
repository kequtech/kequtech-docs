# Introduction

Welcome to KequTech Docs! This is your central hub for documentation on your favorite libraries.

## What is KequTech Docs?

KequTech Docs is a static documentation website that provides comprehensive guides, tutorials, and reference materials for various libraries and tools.

## Features

- **Easy Navigation**: Quickly find the information you need with our intuitive sidebar
- **Search**: Powerful search functionality to locate content instantly
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Dark Mode**: Easy on the eyes with automatic dark mode support

To get started with the documentation, check out the [Getting Started](/guide/getting-started) guide.
